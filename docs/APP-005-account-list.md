## 목적

홈 화면 역할. 저장된 계좌를 즐겨찾기/히스토리 탭으로 보여주고, 빠른 복사·관리 제공.

## UI 구성

```
┌──────────────────────────┐
│  Account Snap     [⚙️]   │
│                          │
│ [⭐ 즐겨찾기][🕒 히스토리]│  ← 탭
│                          │
│ ┌──────────────────────┐ │
│ │ 본가 김치 계좌    📋 │ │  ← 카드: 라벨/탭=복사
│ │ 국민은행             │ │
│ │ 110-1234-5678-9012   │ │
│ └──────────────────────┘ │
│ ┌──────────────────────┐ │
│ │ 우리 가게         📋 │ │
│ │ 신한은행             │ │
│ │ 110-9876-5432-1098   │ │
│ └──────────────────────┘ │
│                          │
│                  [ 📷 ]  │  ← FAB → 카메라
└──────────────────────────┘
```

## 기능

### 즐겨찾기 탭
- `isFavorite: true` 만 표시
- 정렬: 수동 정렬 가능 (long-press drag) 또는 `lastUsedAt desc`
- 카드 탭 = 계좌번호 복사 + `lastUsedAt` 갱신
- swipe 액션: 편집 / 즐겨찾기 해제 / 삭제

### 히스토리 탭
- 전체 (즐겨찾기 포함, 최근 30일 또는 전체)
- 정렬: `createdAt desc`
- 같은 계좌번호는 그룹화 (가장 최근 1개만 표시)
- swipe 액션: 즐겨찾기 추가 / 삭제

### 빈 상태
- 즐겨찾기 비었을 때 → "자주 쓰는 계좌를 즐겨찾기에 추가해보세요" + 카메라 버튼
- 히스토리 비었을 때 → "첫 계좌를 찍어보세요" + 카메라 버튼

### FAB
- 어느 탭에서나 카메라 화면([[APP-002]])으로 이동
