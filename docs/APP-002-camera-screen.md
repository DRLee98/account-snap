## 목적

vision-camera로 계좌 사진 촬영. 결과 이미지를 [[APP-003]] 크롭 화면으로 전달.

## UI 구성

```
┌──────────────────────────┐
│ [×]              [⚡flash]│  ← 상단 컨트롤
│                          │
│                          │
│    카메라 프리뷰          │
│    (전체 화면)            │
│                          │
│                          │
│  ┌──────────────────┐    │
│  │ 계좌가 잘 보이게  │    │  ← 가이드 텍스트
│  │ 촬영해주세요      │    │
│  └──────────────────┘    │
│                          │
│      [ 📷 셔터 ]         │  ← 하단 셔터
│         [갤러리]          │  ← 갤러리에서 선택
└──────────────────────────┘
```

## 기능

- 카메라 권한 요청 (없으면 안내 화면 + 설정 이동 버튼)
- 후면 카메라 기본, 셔터 1회 = 사진 캡처
- 플래시 토글 (auto/on/off)
- 갤러리에서 사진 선택 옵션 (이미 찍어둔 사진 처리)
- 캡처 → 임시 파일 경로 받아 navigation으로 CropScreen에 전달

## 위젯 진입점 처리

위젯에서 카메라로 바로 진입할 수 있어야 함 ([[WGT-004]]). Deep link로 이 화면을 initial route로 띄움.

## 에러 처리

- 카메라 권한 거부 → 안내 화면 + iOS 설정/Android 설정 이동 버튼
- 카메라 디바이스 없음 (시뮬레이터/태블릿) → 갤러리 선택만 활성화
