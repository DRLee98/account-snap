# 스냅넘버 개인정보처리방침

최종 업데이트: 2026-06-02

## 1. 수집하는 정보

스냅넘버는 사용자의 계좌 정보를 외부 서버에 저장하지 않습니다.
모든 계좌 정보(은행명, 계좌번호, 라벨)는 사용자 기기 내 로컬 저장소(MMKV)에만 저장됩니다.

## 2. 외부 서비스 사용

### 2-1. OCR (계좌번호 인식)
계좌번호 인식을 위해 Naver Cloud Platform CLOVA OCR API에 사용자가 촬영한 이미지를 전송합니다.

- 전송 대상: Naver Cloud Platform (한국)
- 전송 데이터: 사용자가 선택한 이미지
- 보관 정책: 처리 즉시 폐기 (NCP CLOVA OCR 약관 기준)

### 2-2. 광고 (Kakao AdFit)
앱 내 배너 광고는 카카오 AdFit이 제공하며, 다음 정보가 광고 서비스에 사용될 수 있습니다.

- 광고 식별자 (IDFA / Advertising Identifier) — 사용자가 ATT 권한을 허용한 경우에 한함
- 기기 정보 (OS 버전, 기기 모델 등)
- 광고 성과 측정용 익명 데이터 (노출/클릭)

ATT(App Tracking Transparency) 권한을 거부해도 앱은 정상 동작하며, 비맞춤형 광고가 표시됩니다.

광고 관련 자세한 처리 방침은 [카카오 AdFit 개인정보처리방침](https://www.kakao.com/policy/privacy)을 참고해주세요.

## 3. 권한 사용

- **카메라**: 계좌 사진 촬영
- **사진 보관함**: 기존 사진에서 계좌 추출
- **알림**: 위젯 탭 시 복사 피드백
- **앱 추적**: 맞춤형 광고 제공 (선택, 거부 가능)

## 4. 데이터 삭제

앱 삭제 시 모든 로컬 데이터가 함께 제거됩니다.
